Basics
======
JSON REST API
  * GET   /api/game/<gid>                     Return the State of the Game (Polling)
  * POST  /api/game                           Add a new Game.
  * POST  /api/game/<gid>/user/               Add a new User to a Game.
  * GET   /api/game/<gid>/user/<uid>/dice     Roll a given number of dice
